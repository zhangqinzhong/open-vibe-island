# References

This directory is reserved for durable external references that the repository wants to keep close at hand.

## Intended Contents

- product behavior baselines
- pinned external articles or screenshots
- reverse-engineering notes that should stay stable across multiple execution plans

## Rules

- keep references focused and durable
- summarize copyrighted material instead of copying it wholesale
- link back to the original source whenever possible
- use this directory for long-lived background material, not for temporary task notes
