# Open Island 手动测试计划

> 版本：v1.0  
> 日期：2026-04-05  
> 目的：在所有 PR 合并后，对 app 进行全面的手动功能验证。

## 测试环境准备

- macOS 14+ (带 notch 的 MacBook 优先，有条件再补测外接屏)
- 已安装 Terminal.app 和/或 Ghostty
- 已安装 Claude Code (`claude`)
- 已安装 Codex (`codex`)
- 构建并运行 Open Island：`zsh scripts/launch-dev-app.sh`

---

- **P0** — 核心链路，必须通过
- **P1** — 重要功能，应当通过
- **P2** — 补充验证，有时间再测
