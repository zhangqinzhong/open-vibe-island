# Pickaxe Swing Preview

Open [pickaxe-swing-preview.html](/Users/wangruobing/Personal/vibe-island/docs/review/pickaxe-swing-preview.html) to review the Terraria-like mining motion direction.

What this file shows:

- `Pebble`, `Mochi`, and `Scout`
- one shared `pickaxe swing` animation family
- one larger preview per mascot
- one tiny icon check per mascot
- one ambient strip check per mascot

Rules:

- mascot body stays on a fixed `8x8` grid
- the pickaxe is a separate pixel layer on the same square grid
- the rock target is static
- impact chips only appear on the hit frame

This is intentionally more playful and game-like than the spark motion family.
